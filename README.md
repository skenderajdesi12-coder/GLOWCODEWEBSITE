# Equalia - Global Women's Community Platform

## About Equalia

Equalia is a safe and welcoming online space for women from all around the world. It's a global community where women can connect, support each other, share experiences, and stay informed about women's issues around the world.

## Features

- **Home Page**: Hero section with mission statement and call-to-action buttons
- **Latest Women's News**: Curated news about women's rights, education, health, and activism
- **International Chatroom**: Real-time community chat with global women
- **Emergency Support**: Crisis resources and international helplines
- **Responsive Design**: Fully mobile-friendly and accessible

## Design Philosophy

Equalia features a modern, elegant design with:
- Feminine color palette (light pink, hot pink, soft white, lavender)
- Rounded corners and smooth animations
- Professional yet empowering aesthetic
- Accessible for all users

## Technologies Used

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (Vanilla)
- Responsive Design
- SVG Icons & Illustrations

2. Open `index.html` in your browser
3. Explore all pages via the navigation bar
